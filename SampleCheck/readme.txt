This sample contains two evaluators and one setup command.
You can see how they are configured in the DependencyChecker.exe.config.

In order to execute this sample, first compile the assembly. 
Then open the location where the assembly was output.
This is typically \bin\Debug\
This folder should include the following:
* DependencyChecker.exe
* DependencyChecker.exe.config
* SampleCheck.dll

Simply execute DependencyChecker.exe to see the sample in action.
